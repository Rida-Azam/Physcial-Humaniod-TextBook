# Physical AI & Humanoid Robotics Textbook

AI-Native textbook with RAG chatbot, multilingual support, and interactive features for teaching Physical AI and Humanoid Robotics.

## Features

- **AI-Native Textbook**: Interactive learning with RAG-powered chatbot
- **Multilingual Support**: English, Urdu, and Roman Urdu
- **Personalization**: Customizable learning experience
- **Interactive Content**: Live code blocks, quizzes, and progress tracking
- **RAG System**: Retrieval Augmented Generation with Qdrant vector database
- **OpenAI Integration**: Advanced AI agents and tools
- **ROS 2 & Robotics**: Comprehensive coverage of Physical AI and Humanoid Robotics

## Getting Started

### Frontend (Docusaurus)

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

### Backend (FastAPI)

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
pip install fastapi uvicorn python-multipart asyncpg qdrant-client openai
```

3. Set environment variables:
```bash
export OPENAI_API_KEY="your-openai-api-key"
export QDRANT_URL="your-qdrant-url"
export NEON_CONNECTION_STRING="your-neon-connection-string"
```

4. Run the server:
```bash
uvicorn app.main:app --reload
```

## Technologies Used

- **Frontend**: Docusaurus v3, React, TypeScript

- **Databases**: Neon PostgreSQL, Qdrant Cloud

- **Authentication**: Better Auth
